Git 常用命令

Git 常用命令

Git 是一个分布式版本控制以及源代码管理工具，它可以为你的项目保存若干快照，以此来对整个项目进行版本管理。

init

初始化一个新的 Git 版本库，这个版本库的配置、存储等信息会被保存到当前目录的.git 文件夹中。

config

修改 Git 配置，可以是版本库的配置，也可以是系统或全局配置。

更多设置

help

Git 帮助文档，可以供我们快速查阅。

status

显示当前工作空间的状态信息。

add

添加文件到当前工作空间。

branch

管理分支，可以通过命令对分支进行增删改查。

checkout

clone

commit

diff

显示当前工作区和提交的不同。

grep

在版本库中进行快速查找。

log

显示版本库的提交日志。

merge

合并分支。

mv

重命名或移动一个文件。

pull

从远程库中拉取代码。

push

rebase

reset
